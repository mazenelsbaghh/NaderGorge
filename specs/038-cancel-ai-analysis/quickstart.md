# User Testing Quickstart: Cancel AI Analysis

1. Pick any YouTube video, upload using its `11` character video ID (e.g., `lQSFol67eW0`).
2. Click the Action item **Generate AI**.
3. It will convert to the `AIProgressTracker`.
4. A red [Cancel X] button will appear below the progress bar.
5. Click **Cancel X** while it says `استخراج الصوت...` or `تحليل Gemini...`.
6. Notice the tracker disappears and the job is successfully aborted.
