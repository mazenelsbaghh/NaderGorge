# Quickstart: Brand Identity Migration

## Setup & Run

1. Start the application locally:
   ```bash
   make dev
   ```

2. Open the landing page in your browser:
   ```
   http://localhost:8738
   ```

3. Verification Steps:
   - Check that the tab title shows "Massar Academy | مسار أكاديمي".
   - Check that the header logo matches the new `logo.svg`.
   - Verify that primary buttons and text headings use Montserrat/Tajawal fonts and Deep Navy/Teal colors.
